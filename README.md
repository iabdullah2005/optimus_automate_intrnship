# Project 1: House Price Prediction

## Business Problem

Accurate house price prediction is important for buyers, sellers, and real estate businesses. This project aims to estimate house prices based on various housing characteristics using machine learning regression techniques.

## Dataset

California Housing Dataset

## Project Workflow

### Data Preparation

* Data Exploration and Analysis
* Missing Value Inspection
* Outlier Detection and Handling
* Feature Selection

### Model Development

Two regression models were trained and evaluated:

1. Linear Regression
2. Random Forest Regressor

### Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

## Outcome

The Random Forest Regressor achieved superior predictive performance compared to Linear Regression and produced more accurate house price estimates.

# Project 2: Customer Churn Prediction

## Business Problem

Customer retention is a major challenge for businesses. Predicting customer churn allows organizations to proactively identify customers at risk of leaving and implement retention strategies.

## Dataset

Telco Customer Churn Dataset

## Project Workflow

### Data Preprocessing

* Missing Value Handling
* Categorical Feature Encoding
* Feature Selection

### Class Imbalance Handling

* SMOTE (Synthetic Minority Oversampling Technique)

### Model Development

Model Used:

* Random Forest Classifier

### Model Evaluation

Performance was evaluated using:

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

### Feature Analysis

Feature importance analysis was performed to identify the most influential factors contributing to customer churn.

## Outcome

The model successfully classified customers likely to churn and demonstrated strong predictive performance after balancing the dataset using SMOTE.
